# Sprint 00 — Foundation

## Goal
Set up the foundational structure and standards for the AI Job Intelligence Platform.

## Single-response tasks
1. Create the initial project folder structure for agent, connector_sdk, plugins, registry, services, tools, models, storage, config, tests, and docs.
2. Add Python project configuration with Python 3.12+, dependency management, and basic development tooling.
3. Create the base Pydantic models for jobs, companies, settings, and plugin metadata.
4. Create the initial TinyDB storage layout and startup helpers for companies, jobs, settings, and plugin metadata.
5. Add configuration loading, logging setup, and environment variable handling.
6. Define coding standards, linting rules, and testing conventions for the repository.
7. Create the initial README, architecture overview, and ADR placeholders.

## Deliverables
- Repository skeleton is ready.
- Core models and storage structure exist.
- Configuration, logging, and documentation foundations are in place.

## Definition of Done
- The project can be opened and understood from the repository structure.
- Core foundational files exist and are documented.
- Future sprints can build on this baseline.
