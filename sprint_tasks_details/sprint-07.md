# Sprint 07 — Google ADK Orchestration

## Goal
Connect the platform to a conversational Google ADK experience.

## Single-response tasks
1. Create the Google ADK agent entry point and basic agent configuration.
2. Implement agent tools for company registration and job search.
3. Connect the agent to the plugin registry and search planner.
4. Define the conversation flow for registration and search requests.
5. Add response formatting for end-user output.
6. Add basic agent-level tests for the main user flows.
7. Document the agent workflow and expected user interactions.

## Deliverables
- A user can interact with the system through the ADK agent.
- The agent can register companies and run searches.

## Definition of Done
- The ADK agent is integrated with core platform workflows.
- Main user flows are working and documented.
- Basic tests exist for the agent layer.
